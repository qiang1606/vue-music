# vue-music
基于 vue2 + vuex 构建一个大型单页面应用仿QQ音乐播放器
## 技术栈
### HTML5 + CSS3 + axios + jsonp + vue2.0 + vuex + vue-router + webpack + ES6
## 项目运行
``` 
git clone https://github.com/qiang1606/vue-music.git  

cd vue-music

npm install

npm run dev

```
